#  Signal-Car-Game-using-HTML-CSS-And-JavaScript

Signal Game is a simple web-based car game that simulates traffic light signals. The game allows the user to control a car using traffic light buttons (Stop, Ready, Go) and a reset button to reset the car's position.

## Features

- *Traffic Light Control*: Three buttons to simulate traffic light signals:
  - *Stop*: Displays the red light and stops the car.
  - *Ready*: Displays the orange light.
  - *Go*: Displays the green light and moves the car.
  
- *Car Movement*: 
  - When the green light is displayed, the car moves from left to right across the screen.
  - The car stops when the red light is displayed.

- *Reset Car*: A reset button to reset the car's position to the starting point.

## Specialties

- *Realistic Traffic Light Simulation*: The game uses images to represent traffic lights, providing a realistic simulation of traffic signals.
- *Simple and Intuitive UI*: The user interface is simple and easy to use, with clearly labeled buttons for controlling the game.
- *Responsive Design*: The game is designed to be responsive and works well on different screen sizes.
- *Smooth Car Movement*: The car moves smoothly across the screen when the green light is displayed, enhancing the gaming experience.

## How to Play

1. *Start the Game*: Open the index.html file in your browser.
2. *Control the Traffic Light*:
   - Click the *Stop* button to display the red light and stop the car.
   - Click the *Ready* button to display the orange light.
   - Click the *Go* button to display the green light and move the car.
3. *Reset the Car: Click the **Reset* button to reset the car to its starting position.

## Project Structure

- *index.html*: The main HTML file that contains the structure of the game.
- *style.css*: The CSS file that styles the game.
- *script.js*: The JavaScript file that contains the game logic.
